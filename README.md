# GitHub User Finder

**Автор:** Атнагулова Альбина Ирековна
**Дата создания:** Апрель 2026
## Описание

GitHub User Finder — это графическое приложение на Python для поиска пользователей GitHub через официальное API. Позволяет искать пользователей, просматривать их профили и сохранять избранных в локальное хранилище JSON.

## Как использовать API

Приложение использует публичное GitHub REST API:
GET https://api.github.com/search/users?q={username}

### Параметры запроса:
- `q` — строка поиска (логин пользователя)
- `per_page` — количество результатов (максимум 100, по умолчанию 20)

### Пример ответа API:
```json
{
  "total_count": 1,
  "items": [
    {
      "login": "octocat",
      "avatar_url": "https://avatars.githubusercontent.com/u/583231?v=4",
      "html_url": "https://github.com/octocat"
    }
  ]
}
```
## Лимиты:
Без аутентификации — 60 запросов в час

С аутентификацией — 5000 запросов в час

## Установка и запуск
Требования:
Python 3.6+

## Запуск (без дополнительных библиотек):
bash
python github_user_finder.py
# Пример использования
## Сценарий: Поиск разработчиков и сохранение избранных
##Шаг 1. Запуск приложения

Откройте программу. Вы увидите окно с полем поиска и двумя пустыми таблицами.

## Шаг 2. Поиск пользователя

В поле «Имя пользователя» введите octocat

Нажмите «Поиск» или клавишу Enter

Результат: В верхней таблице появился пользователь octocat со ссылкой на профиль.

## Шаг 3. Поиск нескольких пользователей

Введите torvalds → Поиск

В таблице появится torvalds (Linus Torvalds)

## Шаг 4. Добавление в избранное

Выберите строку с пользователем octocat

Нажмите «★ Добавить в избранное»

Результат: octocat появился в нижней таблице «Избранные пользователи».

## Шаг 5. Добавление ещё одного в избранное

Найдите torvalds

Выберите его и нажмите «★ Добавить в избранное»

Результат: В избранном теперь два пользователя.

## Шаг 6. Поиск с частичным совпадением

Введите cat → Поиск

Результат: Найдутся все пользователи, в логине которых есть «cat» (например, octocat, cat, scatcat и др.)

## Шаг 7. Открытие профиля из результатов поиска

Выберите любого пользователя в верхней таблице

Нажмите «Открыть профиль»

Результат: Открывается страница пользователя на GitHub.com в браузере.

## Шаг 8. Открытие профиля из избранного

Выберите пользователя в нижней таблице «Избранные»

Нажмите «Открыть профиль»

Результат: Профиль открывается в браузере.

## Шаг 9. Удаление из избранного

Выберите пользователя в нижней таблице

Нажмите «✖ Удалить из избранного»

Результат: Пользователь удаляется из списка избранных.

## Шаг 10. Проверка сохранения данных

Добавьте нескольких пользователей в избранное

Закройте приложение

Откройте снова

Результат: Все избранные пользователи сохранились и отображаются в нижней таблице.
